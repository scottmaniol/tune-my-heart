- LSB (Legacy Standard Bible) ✅ Implemented

## Instructions for Adding New Translations

To add a new Bible translation to the app:

1. **Fill in this template** with all 52 verses in the new translation
2. **Update the code** in `src/data/memoryVerses.ts`:
   - Add a new array (e.g., `memoryVersesNIV`)
   - Add the translation to the `memoryVersesByTranslation` object
3. **Update the translation selector** in `src/components/reading/TranslationSelector.tsx`:
   - Add the new translation to the `TRANSLATIONS` array
4. **Update the type definition** in `src/types/curriculum.ts` if needed

## Tips for Finding Verse Text

- Use BibleGateway.com to look up verses in different translations
- Copy the exact text without verse numbers
- Preserve line breaks for poetic sections (Psalms, etc.)
- Include punctuation exactly as it appears
- For multi-verse selections, include all verses together with appropriate spacing
